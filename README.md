# 📰 GNews Live Dashboard

A modern and responsive web application that fetches real-time news using the GNews API. This app allows users to browse the latest headlines, explore news by categories, and stay updated with global events.

## 🚀 Features

- 🔴 Live news fetching using GNews API
- 📂 Category-based filtering (Technology, Sports, Business, etc.)
- 🔍 Search functionality
- 📱 Fully responsive design
- ⚡ Fast and dynamic UI
